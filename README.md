# Personal-chatbot

This is a beginner-friendly chatbot project built using Python.

Features:
Takes user input
Responds to basic questions
Uses rule-based responses
Exit command using "bye"
Personalized welcome message

Concepts Used:
Functions
Loops
Dictionaries
Conditions
String methods
User input handling

This project helped me improve:
Python fundamentals
Debugging skills
Logical thinking
Understanding of functions and loops
